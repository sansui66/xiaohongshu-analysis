# 小红书爆款分析项目

## 项目说明
这是一个用于分析小红书爬虫数据的项目，主要用于分析爆款内容特征。

## 功能特点
- 数据清洗和预处理
- 爆款特征分析
- 数据可视化展示

## 文件说明
- `data/`: 存放数据文件
- `analysis/`: 数据分析代码
- `git_guide.md`: Git使用指南
- `requirements.txt`: 项目依赖包

## 使用方法
1. 下载项目到本地
2. 安装所需的Python包
3. 运行分析脚本
